2. Formulas — who does what, by what

Each method: who (model), what (predicts what, fitted how), formula, by what (parameters/inputs).

Logistic Regression

Who: Binary/multiclass classifier.
What: Predicts P(Y=1|X). Fitted by maximum likelihood (MLE).
Formula: log P(Y=1|X) / (1 − P(Y=1|X)) = β₀ + β₁x₁ + … + βₚxₚ ⇒ P(Y=1|X) = 1 / (1 + exp(−(β₀ + β′x))).
By what: Coefficients β (one per feature); SE(β̂), Z = β̂/SE(β̂), p-value for each.

Linear Discriminant Analysis (LDA)

Who: Classifier with linear decision boundary.
What: Predicts class by maximizing discriminant δₖ(x). Assumes same covariance Σ for all classes.
Formula: δₖ(x) = x′Σ⁻¹μₖ − (1/2)μₖ′Σ⁻¹μₖ + log πₖ. Choose class k that maximizes δₖ(x).
By what: Class means μₖ, shared Σ, prior πₖ; threshold on P(class|X) for ROC.

Quadratic Discriminant Analysis (QDA)

Who: Classifier with quadratic decision boundary.
What: Like LDA but class-specific covariance Σₖ.
Formula: δₖ(x) uses Σₖ (different per class); decision boundary is quadratic in x.
By what: Class means μₖ and covariances Σₖ, prior πₖ.

Naïve Bayes (Gaussian)

Who: Probabilistic classifier.
What: P(Y|X) ∝ P(Y) ∏ⱼ P(xⱼ|Y). Features assumed independent given class.
Formula: P(xⱼ|Y) = Gaussian(μⱼₖ, σⱼₖ). Class chosen by argmax P(Y) ∏ⱼ P(xⱼ|Y).
By what: Per-class mean and variance for each feature.

Linear regression (OLS)

Who: Regression for continuous/count outcome.
What: E[Y|X] = Xβ; fitted by minimizing sum of squared residuals.
Formula: β̂ = (X′X)⁻¹X′y; predictions ŷ = Xβ̂.
By what: Coefficients β; RMSE on test set.

Poisson regression (GLM)

Who: Regression for count outcome.
What: log E[Y|X] = Xβ; Var(Y) ∝ E[Y]. Fitted by MLE (Poisson likelihood).
Formula: E[Y|X] = exp(Xβ); keeps predictions non-negative.
By what: Coefficients β; RMSE on test set; preferred for counts.
